# 2. 스프링 핵심 원리 - 기본편

### 개발 환경

-   java version 17
-   spring boot version 3.x.x

### gradle spring boot 실행

```
./gradlew bootRun --continuous

```

### gradle spring build

```
./gradlew build

# 테스트 제외하고 빌드
./gradlew build -x test

```

### gradle dependencies sync.

```
./gradlew dependencies
```

### h2 실행

```
./h2.sh

# window
./h2.bat

```

### swagger 주소

```
{host}/swagger-ui/index.html
{host}/swagger-ui
```
